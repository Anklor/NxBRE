NxBRE DSL
=========

The solution contains 3 projects:

- NxDSL: this library is the one to use in your application, it contains the Adapter to use to load the DSL rule base.

- Examples: demonstrates the usage of the NxDSL adapter.

- GUI: this simple read-only GUI allows you to check an English DSL file and see if there is a parsing issue and what it is.
